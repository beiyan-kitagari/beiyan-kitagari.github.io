NVM：安装多个版本的NPM，并在不同版本之间切换的命令行工具

2020-04-26

NVM：複数バージョンのNPMをインストールして、便利的にバージョンを切り替えるコマンドラインツール
NVM：A command line tool can install multiple version of NPM and do management of them

GitHub: https://github.com/nvm-sh/nvm

官方提供了方便的安装脚本，可以在终端下使用curl或wget来获取并进行安装。
開発者の方は便利なインストーラスクリプトを提供したので、ターミナルでcurlまたはwgetを使ったらそれをダウンロードして、インストール出来ます。
The develop team of this tool released a convenient shell script,you can use curl or wget in termial to get and run this script to install NVM.

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash

当前展示的版本号为0.35.3，使用的方法是curl，安装时请不要直接复制上面的命令，请访问官方的Github项目页面，依照readme.md的提示安装最新的版本。
今のサンプルコマンドのバージョンは0.35.3で、インストール方式はcurlです。インストールする時に上記のコマンドを直接コピーしなくて、GitHubのオフィシャルページにあるReadme.mdファイルを参照して、最新版をインストールしてください。
The example of install command the 0.35.3 version, download by curl, please don't directly copy the sample command, and go to the official page on GitHub to get the latest version.

安装结束之后，使用nvm --version进行验证，能看到版本号的话证明安装成功。
インストールが完成したら、nvm --versionを実行して、バージョン情報を見えるなら、成功です。
You can run nvm --version after the install process, if will return the version code, if it was installed correctly.

你可以使用如下命令安装最新版的node。
下記のコマンドを実行したら、最新版のnodeをインストールできます。
You can use this command to install the latest version of node.

nvm install node

如果要安装特定版本的node，可以使用这样的命令。
特定バージョンのnodeをインストールしたいなら、以下のコマンドを実行してください。
If you want to install a specific version of node, you can run this command.

nvm install 11.6.0

使用nvm list命令，可以看到现在安装过的所有node版本。
nvm listコマンドを使ったら、システムにインストールされた全てのnodeバージョンを見えます。
You can use this command nvm list to check all installed versions of node.

使用nvm install --lts，可以安装最新的LTS版本node。
nvm install --ltsを使ったら、nodeの最新のLTSバージョンをインストールできます。
You can install the lastest LTS version of node with run nvm install --lts

使用nvm use --lts 可以将系统默认的node环境切换到最新的LTS版本，将后面的--lts换成版本号的话可以切换到指定的版本号，如nvm use 11.6.0。
nvm use --ltsを使ったら、システムのデフォルトnode環境を最新のLTSバージョンへ切り替えます。後ろの「--lts」をバージョン番号に変えたら、例えばnvm use 11.6.0を実行したら、11.6のバージョンに切り替えます。
You can switch to the latest LTS verison with nvm use --lts, or use nvm use 11.6.0 to switch to a spiecific version.
